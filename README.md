# kintone UI Component デモ

[![Ask DeepWiki](https://deepwiki.com/badge.svg)](https://deepwiki.com/r3-yamauchi/kintone-ui-component-sample-plugin)

kintone UI Component の様々なコンポーネントを体験できるデモプラグインです。

- **デスクトップ版**: プラグイン設定画面で19種類のデスクトップコンポーネントを体験
- **モバイル版**: レコード一覧画面で10種類のモバイルコンポーネントを体験

## 機能

### デスクトップ版コンポーネント（19種類）

このプラグインでは、以下の19種類のkintone UI Component（デスクトップ版）を体験できます：

### ファイル・添付

1. **Attachment** - ファイル添付コンポーネント
   - ファイル選択機能
   - 選択したファイルの情報を表示

### 基本操作

2. **Button** - ボタンコンポーネント
   - 通常ボタン、送信ボタン、無効なボタンの3種類を表示
   - クリックイベントの動作を確認できます

### 選択系

3. **Checkbox** - チェックボックスコンポーネント
   - 複数選択が可能
   - 趣味選択のサンプル

4. **Combobox** - コンボボックスコンポーネント
   - 検索可能なドロップダウンリスト
   - 果物のサンプルデータで選択を体験

### 日付・時刻系

5. **DatePicker** - 日付選択コンポーネント
   - カレンダーから日付を選択
   - 選択した日付をリアルタイムで表示

6. **DateTimePicker** - 日付時刻選択コンポーネント
   - 日付と時刻を同時に選択
   - リアルタイムで選択値を表示

### 選択系（続き）

7. **Dropdown** - ドロップダウンコンポーネント
   - 単一選択のドロップダウン
   - 都道府県選択のサンプル

### レイアウト系

8. **FieldGroup** - フィールドグループコンポーネント
   - 複数のフィールドをグループ化
   - Text、Dropdownなどの入力コンポーネントを含む

### 選択系（続き）

9. **MultiChoice** - 複数選択コンポーネント
   - チップ形式で複数選択
   - プログラミング言語選択のサンプル

10. **RadioButton** - ラジオボタンコンポーネント
    - 単一選択のラジオボタン
    - 配送方法選択のサンプル

### データ表示

11. **ReadOnlyTable** - 読み取り専用テーブルコンポーネント
    - データをテーブル形式で表示
    - サンプルの社員情報を表示

### トグル・スイッチ

12. **Switch** - スイッチコンポーネント
    - オン/オフを切り替え
    - 通知設定などで使用

### データ編集

13. **Table** - テーブルコンポーネント
    - 編集可能なテーブル
    - 行の追加/削除が可能
    - セル値の変更を検知

### レイアウト系（続き）

14. **Tabs** - タブコンポーネント
    - コンテンツをタブで切り替え
    - 3つのタブサンプル

### 入力系

15. **Text** - テキスト入力コンポーネント
    - 1行テキスト入力
    - リアルタイムで入力値を表示

16. **TextArea** - テキストエリアコンポーネント
    - 複数行テキスト入力
    - 文字数カウント機能付き

### 日付・時刻系（続き）

17. **TimePicker** - 時刻選択コンポーネント
    - 時刻を選択
    - リアルタイムで選択値を表示

### ヘルプ・ガイド

18. **Tooltip** - ツールチップコンポーネント
    - マウスオーバーで情報を表示
    - ボタンなどの要素に補足説明を追加

### ユーザー選択

19. **UserOrgGroupSelect** - ユーザー/組織/グループ選択コンポーネント
    - kintone環境のグループ情報を取得して表示
    - 複数選択が可能

### モバイル版コンポーネント（10種類）

モバイル版では、以下の10種類のkintone UI Component（モバイル版）を体験できます：

1. **MobileButton** - モバイル用ボタンコンポーネント
   - タップイベントを処理

2. **MobileText** - 1行テキスト入力コンポーネント
   - リアルタイムで入力値を表示

3. **MobileTextArea** - 複数行テキスト入力コンポーネント
   - 文字数カウント機能付き

4. **MobileCheckbox** - チェックボックスコンポーネント
   - 複数選択が可能

5. **MobileRadioButton** - ラジオボタンコンポーネント
   - 単一選択

6. **MobileDropdown** - ドロップダウンコンポーネント
   - 単一選択のドロップダウン

7. **MobileMultiChoice** - 複数選択コンポーネント
   - チップ形式で複数選択

8. **MobileDatePicker** - 日付選択コンポーネント
   - カレンダーから日付を選択

9. **MobileTimePicker** - 時刻選択コンポーネント
   - 時刻を選択

10. **MobileDateTimePicker** - 日付時刻選択コンポーネント
    - 日付と時刻を同時に選択

## セットアップ

### 1. 依存パッケージのインストール

```bash
npm install
```

### 2. 秘密鍵の生成（初回のみ）

```bash
npm run keygen
```

### 3. プラグインのビルド

```bash
npm run build
```

ビルドされた `dist/plugin.zip` が生成されます。

## 使用方法

### デスクトップ版

1. プラグインをインストール後、設定画面を開きます
2. 19種類のコンポーネントが縦に並んで表示されます
3. 各コンポーネントを操作して動作を確認できます：
   - ボタンをクリック
   - 日付や時刻を選択
   - ドロップダウンやコンボボックスから項目を選択
   - チェックボックスやラジオボタンで選択
   - テキストやテキストエリアに入力
   - スイッチでオン/オフを切り替え
   - テーブルに行を追加/削除
   - タブを切り替え
   - ツールチップを表示
   - グループを選択
   - ファイルを添付

### モバイル版

1. プラグインをインストール後、モバイル端末でアプリを開きます
2. レコード一覧画面を表示します
3. ヘッダーメニュースペースに10種類のモバイルコンポーネントが縦に並んで表示されます
4. 各コンポーネントを操作して動作を確認できます：
   - ボタンをタップ
   - テキストやテキストエリアに入力
   - チェックボックスやラジオボタンで選択
   - ドロップダウンから項目を選択
   - 複数選択コンポーネントで選択
   - 日付や時刻を選択

## 開発コマンド

```bash
# プラグインのビルド（ファイル監視モード）
npm run develop

# プラグインのアップロード（kintoneへの自動アップロード）
npm run upload

# ESLintでコードチェック
npm run lint
```

## プロジェクト構成

```
/
├── src/
│   ├── manifest.json          # プラグインマニフェスト
│   ├── html/
│   │   └── config.html        # 設定画面HTML
│   ├── js/
│   │   ├── config.js          # 設定画面JavaScript（19種類のコンポーネントデモ）
│   │   ├── desktop.js         # デスクトップ版JavaScript（最小限の実装）
│   │   ├── mobile.js          # モバイル版JavaScript（10種類のモバイルコンポーネントデモ）
│   │   └── kuc.min.js         # kintone UI Component v1.24.0
│   ├── css/
│   │   ├── config.css         # 設定画面スタイル
│   │   └── 51-modern-default.css  # kintone UI Componentスタイル
│   └── image/
│       └── icon.png           # プラグインアイコン
├── package.json
└── README.md
```
