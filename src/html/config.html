kintone UI Component デモ

このプラグインでは、様々なkintone UI Componentをインタラクティブに体験できます。
各コンポーネントを操作して、動作を確認してみてください。

Attachment 📖 ドキュメント

ファイル添付コンポーネント

Button 📖 ドキュメント

ボタンコンポーネント - クリックイベントを処理できます

Checkbox 📖 ドキュメント

チェックボックスコンポーネント - 複数選択可能

Combobox 📖 ドキュメント

コンボボックスコンポーネント - 検索可能なドロップダウン

DatePicker 📖 ドキュメント

日付選択コンポーネント

DateTimePicker 📖 ドキュメント

日付時刻選択コンポーネント

Dropdown 📖 ドキュメント

ドロップダウンコンポーネント - 単一選択

FieldGroup 📖 ドキュメント

フィールドグループコンポーネント - 複数のフィールドをグループ化

MultiChoice 📖 ドキュメント

複数選択コンポーネント - チップ形式で複数選択

RadioButton 📖 ドキュメント

ラジオボタンコンポーネント - 単一選択

ReadOnlyTable 📖 ドキュメント

読み取り専用テーブルコンポーネント

Switch 📖 ドキュメント

スイッチコンポーネント - オン/オフを切り替え

Table 📖 ドキュメント

テーブルコンポーネント - 行の追加/削除が可能

Tabs 📖 ドキュメント

タブコンポーネント - コンテンツをタブで切り替え

Text 📖 ドキュメント

テキスト入力コンポーネント

TextArea 📖 ドキュメント

複数行テキスト入力コンポーネント

TimePicker 📖 ドキュメント

時刻選択コンポーネント

Tooltip 📖 ドキュメント

ツールチップコンポーネント - マウスオーバーで情報を表示

UserOrgGroupSelect 📖 ドキュメント

ユーザー/組織/グループ選択コンポーネント

注意: これらのコンポーネントはkintone UI Component v1を使用しています。
詳細は kintone UI Component 公式ドキュメント をご確認ください。
